# CLT na Prática — Painel de Direitos Trabalhistas

Aplicativo Streamlit de consulta a direitos trabalhistas (projeto acadêmico).

Arquivos principais:
- `app.py`: aplicação Streamlit que exibe uma base própria de direitos trabalhistas.
- `requirements.txt`: dependências necessárias para executar a aplicação.

Pré-requisitos:
- Python 3.8+ (recomendado)

Como executar (PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

Observações:
- Este repositório é destinado a uso acadêmico e didático; a consulta não substitui
	orientação jurídica individualizada.
- Para atualizar dependências, edite `requirements.txt` e reinstale.
